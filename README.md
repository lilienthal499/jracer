# JRacer

JavaScript Racing Game - My first coding project from 2009-2015!

## Quick Start

```bash
npm install
npm start
```

Open your browser at http://localhost:5500

## Documentation

See the [docs/](docs/) folder for complete documentation:
- **README.md** - Full project history
- **DEVELOPMENT.md** - Development guide
- **TIMELINE.md** - Development timeline

## Structure

- `js/` - Game code
- `css/` - Stylesheets
- `docs/` - Documentation
- `archive/` - Historical versions

## Development

```bash
npm run lint       # Check code
npm run format     # Format code
code jracer.code-workspace  # Open in VS Code
```
